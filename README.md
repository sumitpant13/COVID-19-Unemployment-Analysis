# Unemployment Rate Analysis During COVID-19

## Project Overview
This project analyzes unemployment trends during the COVID-19 pandemic. The goal is to understand how unemployment rates changed over time and across different regions.

## Dataset
The dataset includes unemployment statistics such as region, date, unemployment rate, employment count, and labour participation rate.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed
1. Data loading and cleaning
2. Handling missing values
3. Exploratory data analysis
4. Data visualization
5. COVID-19 trend analysis

## Key Insights
The analysis shows a noticeable increase in unemployment rates during the COVID-19 period.

## Conclusion
The project highlights the economic impact of the COVID-19 pandemic on employment trends.
